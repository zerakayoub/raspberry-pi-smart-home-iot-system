# Automations

The system uses Home Assistant’s automation framework to implement event-driven workflows.

Examples include:

Security Monitoring

Door and motion sensors monitor activity

Alerts sent to mobile devices when activity is detected

Centralized monitoring dashboard

Lighting Automation

Motion sensors trigger lighting in specific areas

Time-based and event-based lighting rules

Activity-based Automation

A vibration sensor installed on a kitchen counter triggers lighting when movement is detected after sunset.
